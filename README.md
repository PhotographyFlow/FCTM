# TO/GA Photos上传指南

本指南由TO/GA Photos图片审核团队编写，旨在帮助您尽可能顺利的完成图片编辑和上传过程。我们希望通过这本上传指南帮助用户更好的理解TO/GA Photos的审核标准，让您的图片尽可能符合TO/GA Photos的标准。

> **[warning] 遵循本指南会让您的图片更加符合TO/GA Photos的标准，不代表图片一定能被接受。**

如果这本指南没有帮助到您，我们欢迎您向TO/GA Photos团队寻求帮助。
